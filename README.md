# Turki Alshaalan - Portfolio Website

A personal portfolio website showcasing the work and skills of Turki Alshaalan, an IT student, Game Developer, and Unity & C# Programmer.

## Overview

This portfolio website serves as a professional presentation of my game development work, technical skills, and professional background. It features a modern, responsive design with a unique pixel-art aesthetic inspired by retro gaming.

## Features

- **Interactive Game**: Built-in playable battle game accessible from the navigation logo
- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Dark/Light Theme**: Toggle between dark and light modes for user preference
- **Sound Effects**: Interactive sound system with toggle functionality
- **Smooth Navigation**: Single-page navigation with smooth scrolling between sections
- **Project Showcase**: Display of game projects with links to itch.io

## Sections

1. **Home** - Hero section with profile picture and introduction
2. **About** - Detailed background information
3. **Projects** - Showcase of game projects
4. **Experience** - Professional work history
5. **Skills** - Technical skills and competencies
6. **Education** - Academic background
7. **Certifications** - Professional certifications
8. **Contact** - Contact information and form

## How to View

1. Clone or download this repository
2. Open `index.html` in any modern web browser
3. Navigate through sections using the navbar
4. Click the logo (play icon) in the navbar to play the built-in game

## Game Controls

- **A/D** - Move left/right
- **W/Space** - Jump
- **J** - Attack
- **K** - Block/Shield
- **ESC** - Exit game

Mobile controls are also available with touch-friendly buttons.
